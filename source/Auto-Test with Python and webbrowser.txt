
python基于浏览器的自动化测试

1.python3
2.安装selenium
3.下载IEDriverServer（与selenium版本号一致），并解压，解压后的文件放入IE的目录
4.参考https://blog.csdn.net/jichuang123/article/details/53008581完成IE设置
5.示例代码：
	# -*- coding: utf-8 -*-
	from selenium import webdriver
	from selenium.webdriver.common.keys import Keys
	
	driver = webdriver.Ie(executable_path='C:\\Program Files\\Internet Explorer\\IEDriverServer.exe')
	driver.get('https://www.baidu.com')
	assert u'百度一下，你就知道' in driver.title
	print ("当前URL：", driver.current_url)
	
	out=driver.page_source		#这是已经经过解析的网页内容，可以送给BeautifulSoup处理了

	driver.quit()		# quit方法关闭整个浏览器
	
	
使用pyttsx3引擎将文本转化为语音

Python3环境
1.安装需要的包：
py -3 -m pip install pyttsx3
py -3 -m pip install pypiwin32	#在Anacanda Prompt环境下，如果使用了Anaconda的话

举例代码
import pyttsx3
engine = pyttsx3.init()
engine.say('Sally sells seashells by the seashore.')
engine.say('The quick brown fox jumped over the lazy dog.')
engine.runAndWait()

更换语音
engine = pyttsx3.init()
voices = engine.getProperty('voices')
for voice in voices:
   engine.setProperty('voice', voice.id)
   engine.say('The quick brown fox jumped over the lazy dog.')
engine.runAndWait()

这里voice.id是已经安装到系统中的TTS语音，如
HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Speech\\Voices\\Tokens\\MS-Lili-2052-20-DSK

如果要增加语音种类，可以到 http://www.microsoft.com/en-us/download/details.aspx?id=27224 下载
增加的语音可能由于windows版本差异的原因找不到新安装的语音库，参考 https://www.cnblogs.com/lerit/p/3486854.html?utm_source=tuicool 这里解决


设置语速
engine = pyttsx3.init()
rate = engine.getProperty('rate')
engine.setProperty('rate', rate+50)			#rate是int型变量，数字越大语速越快
engine.say('The quick brown fox jumped over the lazy dog.')
engine.runAndWait()

调整音量		#这个可以通过系统的音量调节
engine = pyttsx3.init()
volume = engine.getProperty('volume')
engine.setProperty('volume', volume-0.25)
engine.say('The quick brown fox jumped over the lazy dog.')
engine.runAndWait()
